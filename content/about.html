About

Meatspace is a personal project of Indro De. It will grow to contain pages for personal reference, picture galleries, and writings on other nonspecific topics. You may call it a hybrid between a blog, photo gallery, reference manual, and personal shoutbox.

This website will be constantly tweaked and updated while maintaining a simplistic, minimal layout. You may follow the source code on GitHub. A quick introduction on the development of Meatspace can be read right here.

Built using Ruby, Foundation, and nanoc. Powered by nginx on a Linode VPS. This site is valid HTML5 and enhanced for mobile devices. Fonts provided and hosted by Typekit.
